# 🌌 Огляд

**CHIP-8 Emulator** — це ретро-емулятор віртуальної машини CHIP-8, написаний на Python.  
Проект створений як поєднання низькорівневої логіки та естетики старого заліза:

- 🧠 **Низькорівнева емуляція CPU**
- ⚡ **Декодування Opcode в реальному часі**
- 🕹️ **Класичний Retro Gaming**
- 🌃 **Dark Cyberpunk Aesthetics**

# 🧠 Архітектура емулятора

### 🔹 `load_rom()`
Завантажує бінарний код гри в пам'ять за адресою `0x200`.

### 🔹 `cycle()`
Основний цикл процесора: **FETCH** (Зчитування) -> **DECODE** (Декодування) -> **EXECUTE** (Виконання).

### 🔹 `print_debug_info()`
Виводить поточний стан системи: Opcodes, регістри, лічильник команд (PC) та стан стеку.
