# 🌌 Огляд

**CHIP-8 Emulator** — це повнофункціональний ретро-емулятор віртуальної машини CHIP-8, написаний на Python.

Проект створений як поєднання низькорівневої логіки та естетики старого кіберпанк-заліза:

* 🧠 **Низькорівнева емуляція CPU** (Fetch-Decode-Execute)
* ⚡ **Декодування 35 Opcodes у реальному часі**
* 🕹️ **Класичний Retro Gaming**
* 🌃 **Dark Cyberpunk Aesthetics** (Сканлайни, неонове світіння, фосфорний шлейф)
* 🔊 **Синтезований звук** (Математична генерація Square Wave)

# 🎮 Керування

Система використовує сучасну розкладку для емуляції оригінальної 16-кнопкової клавіатури.

### Ігрова розкладка:

| Оригінал CHIP-8 | Клавіатура ПК |
| --- | --- |
| `1 2 3 C` | `1 2 3 4` |
| `4 5 6 D` | `Q W E R` |
| `7 8 9 E` | `A S D F` |
| `A 0 B F` | `Z X C V` |

### Системні команди:

* **`SPACE`** — Пауза / Продовжити гру.
* **`ESC`** — Екстрений вихід із системи.

# 🧠 Архітектура емулятора

### 🔹 `load_rom()`

Завантажує бінарний код гри в пам'ять за адресою `0x200`.

### 🔹 `cycle()`

Основний цикл процесора: **FETCH** (Зчитування) -> **DECODE** (Декодування) -> **EXECUTE** (Виконання).

### 🔹 `print_debug_info()`

Динамічний дизасемблер та система логування. Виводить поточний стан системи: Opcodes, регістри, прапорці зіткнень (VF) у консоль та лог-файл.
